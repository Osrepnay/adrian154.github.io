Compact Object (COB) Format Specification

Introduction

The Compact Object format, or COB, is a file format for 3D models.

Definitions

Datatypes

Name Size (bytes) Description
Boolean 1 0 for true, 1 for false
Byte 1 Signed 8-bit two's complement integer (range of -128 to 127)
Unsigned Byte 1 Unsigned 8-bit integer (range of 0 to 255)
Short 2 Signed 16-bit two's complement integer (range of -32,768 and 32,767)
Unsigned Short 2 Unsigned 16-bit integer (range of 0 to 65535)
Int 4 Signed 32-bit two's complement integer (range of -2,147,483,648 to 2,147,483,647)
Unsigned Int 4 Unsigned 32-bit integer (range of 0 to 4,94,967,295)
Long 4 Signed 64-bit two's complement integer (range of -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807)
